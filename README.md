원하는 확장자(PNG/JPG/WebP)로 쓰셔도 되지만, `index.html`의 파일명과 확장자를 맞춰주세요.
필요하면 `index.html`에서 경로만 바꾸면 됩니다.

1) `images/` 폴더에 위 파일 3개 넣기
